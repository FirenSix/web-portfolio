# Portfólio Pessoal 🚀

Este é um projeto de portfólio pessoal responsivo desenvolvido como parte das atividades práticas do curso de Engenharia de Software. O objetivo principal foi consolidar os conceitos fundamentais de desenvolvimento web no client-side, integrando a estruturação semântica do HTML, a estilização moderna com CSS e a interatividade em tempo real com JavaScript puro.

## 🛠️ Tecnologias Utilizadas

* **HTML5**: Estruturação semântica da página, organização do conteúdo e criação do formulário.
* **CSS3**: Layout adaptável utilizando Flexbox e Grid, Media Queries para garantir a responsividade em dispositivos móveis e estilização completa dos temas.
* **JavaScript (Vanilla)**: Manipulação da DOM para controle dos elementos da tela, validação de segurança do formulário de contato e controle de eventos de clique.
* **Web Storage API (localStorage)**: Utilizada para persistir e memorizar a escolha de tema do usuário no navegador.

## 🌟 Principais Recursos

* **Alternância de Tema (Dark/Light Mode)**: Sistema dinâmico que altera completamente a paleta de cores do site através de um único botão.
* **Persistência de Estado**: O navegador lembra se você selecionou o modo escuro ou claro, mantendo a preferência mesmo se a página for atualizada.
* **Design Responsivo**: Layout 100% adaptado para computadores, tablets e smartphones.
* **Formulário de Contato Inteligente**: Interface que exibe o formulário sob demanda e realiza a validação dos campos e do formato do e-mail antes do envio.
